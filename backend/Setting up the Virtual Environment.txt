Setting up the Virtual Environment

1. cd to the avenudub/backend directory
2. python3 -m venv avenudub_venv
3. . avenudub_venv/bin/activate
4. pip install flask
5. brew install mysql@8.4
6. pip install mysqlclient
7. ln -s /opt/homebrew/opt/mysql@8.4 /opt/homebrew/opt/mysql



(optional) pip install pymysql
